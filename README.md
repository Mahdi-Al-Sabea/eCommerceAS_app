## Before running the app , make sure that you have the backend server ready and running 

## Run the app

1. Clone the repo
   ```bash
   git clone https://github.com/Mahdi-Al-Sabea/eCommerceAS_app.git
   cd eCommerceAS_app
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Update constants/serverIP.js and put your computer's ip address in ip
   

4. Run on Android
   ```bash
   npx react-native run-android
   ```

5. You can create your own user or directly login using seeded user email:user@1 password:password123
